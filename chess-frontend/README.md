# Chess Frontend

React + Three.js (React Three Fiber) 3D chess client with realistic per-piece GLTF models (knight, rook, bishop, etc.).

## Run

1. Start the backend (`chess-backend`) on port 8080.
2. Install and run:

```bash
npm install
npm run dev
```

Open **http://localhost:5173**.

## Configuration

Copy `.env` and set the API URL if needed:

```
VITE_API_URL=http://localhost:8080
```
