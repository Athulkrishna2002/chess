# 3D Chess Piece Models

Each piece uses a dedicated GLTF model (knight looks like a knight, rook like a rook, etc.).

Models are from the [chess3d](https://github.com/Sushant-Coder-01/chess3d) project (Sketchfab **Feld1** chess set).
